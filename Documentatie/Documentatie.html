Purpose

The aim of this project is to help clients find hotels on a specific location and date.

Usage

Interactive interface

In interactive mode, the user is able to enter filtering hotel and make a rezervation.

Implementation

Framework

The foundation of the app is built on Laravel, a web application framework in PHP with flexible RESTful routing and MVC structure.

A basic route looks like this: 

				Route::get('logout', function(){

				Session::forget('username');
				Session::forget('password');
				return Redirect::to('login');
				});
which allows easy creation of templates with pre-populated variables already injected into it.

Structure of users

Admin site - have a special page where can modify/add or delete users, rezervations, hotels, facilities.

Hotel adminitrator - have a special page where can see details about his hotel.

Normal user - can only filter hotels and make rezervations.
